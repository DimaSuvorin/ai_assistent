# Проект: Создание ИИ-ассистента для компании СЕРКОНС

ссылка на проект: @Serconslab_bot

![Скрин_1](https://github.com/DimaSuvorin/ai_assistent/blob/main/Screenshot%202025-04-15%20170037.png?raw=true)


## Описание проекта

Проект направлен на создание интеллектуального ассистента в Telegram, который будет предоставлять консультации по услугам компании СЕРКОНС, управлять расписанием клиентов через интеграцию с Google Calendar и обеспечивать эффективное взаимодействие между клиентами и менеджерами компании. Для реализации проекта используются следующие технологии:
- **Qwen** — крупная языковая модель для обработки естественного языка и предоставления точных ответов.
- **App.suvvy.ai** — платформа для создания и управления диалоговыми интерфейсами.
- **Google Calendar API** — для автоматизации записи клиентов на консультации или встречи.

Цель проекта — упростить процесс взаимодействия клиентов с компанией, повысить конверсию за счет своевременной передачи клиентов менеджерам и автоматизировать рутинные процессы.

---

## Основные функции ассистента

1. **Консультирование по услугам компании СЕРКОНС**  
   - Предоставление информации о сертификации продукции, систем менеджмента, декларировании соответствия и других услугах.
   - Ответы на часто задаваемые вопросы (FAQ) с использованием базы знаний компании.
   - Уточнение потребностей клиента через задавание уточняющих вопросов.

2. **Интеграция с Google Calendar**  
   - Автоматическая запись клиентов на консультации с менеджерами.
   - Проверка доступности времени менеджеров и предложение удобных слотов для встреч.
   - Отправка напоминаний о предстоящих встречах через Telegram.

3. **Перевод на менеджера по продажам**  
   - Передача контактных данных клиента менеджеру после уточнения запроса.
   - Автоматическое создание задачи в календаре менеджера для последующего общения.

4. **Аналитика и отчеты**  
   - Сбор данных о частых запросах клиентов для улучшения качества обслуживания.
   - Формирование отчетов о количестве обращений и успешных переводах на менеджеров.

---

## Технологический стек

### 1. **Языковая модель**
- **Qwen** используется для обработки естественного языка и генерации ответов на вопросы клиентов. Модель обучена на данных компании СЕРКОНС и предоставляет точные и релевантные ответы.

### 2. **Платформа для создания бота**
- **App.suvvy.ai** — используется для разработки диалогового интерфейса и управления логикой взаимодействия с клиентами. Платформа позволяет легко интегрировать Qwen и настраивать сценарии общения.

### 3. **Интеграция с Google Calendar**
- **Google Calendar API** — обеспечивает автоматическую запись клиентов на встречи и управление расписанием менеджеров.

### 4. **Telegram Bot API**
- Используется для создания чат-бота в Telegram, который будет взаимодействовать с клиентами в реальном времени.

---

## Архитектура системы

1. **Клиентский интерфейс**  
   - Telegram-бот принимает сообщения от клиентов и отправляет их на обработку в Qwen.

2. **Обработка запросов**  
   - Qwen анализирует запросы, формирует ответы и определяет дальнейшие действия (например, необходимость записи на встречу).

3. **Интеграция с Google Calendar**  
   - При необходимости записи на встречу данные передаются в Google Calendar API для создания события.

4. **Передача данных менеджерам**  
   - После завершения диалога с клиентом его контактные данные и информация о запросе передаются менеджерам через внутренние системы компании.

## Этапы реализации

1. **Подготовка данных**  
   - Сбор и структуризация информации об услугах компании СЕРКОНС.
   - Создание базы знаний для Qwen.

2. **Разработка бота**  
   - Настройка диалоговых сценариев в App.suvvy.ai.
   - Интеграция Qwen для обработки запросов.

3. **Интеграция с Google Calendar**  
   - Настройка API для автоматической записи клиентов на встречи.

4. **Тестирование**  
   - Проведение тестов для проверки корректности работы бота и интеграции с Google Calendar.

5. **Запуск и мониторинг**  
   - Развертывание бота в Telegram.
   - Мониторинг работы и сбор обратной связи для дальнейшего улучшения.

## Преимущества проекта

- **Автоматизация рутинных процессов**: Уменьшение нагрузки на менеджеров за счет автоматической обработки запросов.
- **Улучшение качества обслуживания**: Быстрые и точные ответы на вопросы клиентов.
- **Повышение конверсии**: Эффективный перевод клиентов на общение с менеджерами.
- **Гибкость и масштабируемость**: Возможность добавления новых функций и расширения базы знаний.
